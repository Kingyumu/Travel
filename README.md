# Travel
一个前后端分离的JavaWeb项目 --- 黑马旅游网

### 技术栈
Java + Mysql + Servlet + Html/Css/Jquery + Redis + Tomcat8

### 工具
IntelliJ IDEA + Maven

### 主要功能
注册 + 登录 + 邮箱验证/激活 + 展示线路列表 + 查询线路详情 + 点击收藏
